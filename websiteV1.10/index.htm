Online Portal for High-Precision Atomic Physics Data and Computation

This portal provides a wide range of transition matrix elements, static, and dynamic polarizabilities for a number of atoms and ions, including Li, Be+, Na, Mg+, K, Ca+, Rb, Sr+, Sr, Ba+, Fr, Ra+, and others. The data are calculated using a high-precision state-of-the-art coupled-cluster (CC) method or a hybrid method that combines configuration interaction and CC. All values include estimated uncertainties. Experimental values are also included with references where high-precision data are available. We seek community input to improve the portal and guide the next stages of the project which will include more complicated systems and capabilities to compute atomic properties on demand via portal.
